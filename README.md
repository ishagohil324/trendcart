TrendCart - E-Commerce Platform

A modern e-commerce web application built with React and Firebase, featuring authentication, product browsing, cart management, and secure checkout.

🚀 Features

🔐 User authentication (Email/Password & Google)

🛒 Add to cart & wishlist

🔍 Product search & category filtering

💳 Secure checkout (Razorpay integration)

📦 Order tracking & history

📊 User dashboard

📱 Fully responsive UI

🛠 Tech Stack

Frontend: React.js, Tailwind CSS

Backend/Services: Firebase (Auth + Firestore)

Payments: Razorpay

Other: React Router, Context API, Framer Motion
